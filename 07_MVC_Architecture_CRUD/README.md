# MVC Architecture with CRUD Operations in Express.js

## 📖 About

This folder demonstrates the implementation of a complete **CRUD (Create, Read, Update, Delete)** application using the **Model-View-Controller (MVC)** architecture in Express.js. The application follows a structured project architecture while interacting with MongoDB through Mongoose to manage user data efficiently.

- Read Operation
- Update Operation

- Express Routing
- Mongoose Models

## 📌 CRUD Operations

### Create
Adds a new user to the MongoDB database.

### Read
Retrieves and displays all users stored in the database.

### Update
Allows existing user information to be modified and updated in MongoDB.

### Delete
Removes a selected user from the database.
