# JWT User Authentication in Express.js

## 📖 About

This folder demonstrates the implementation of **User Authentication** using **JSON Web Tokens (JWT)** in Express.js. The application enables users to register, log in securely, and access protected routes using JWT-based authentication while following the MVC architecture.

---

## 🚀 Concepts Covered

- User Registration
- User Login
- JWT Token Generation
- JWT Verification
- Protected Routes
- Authentication Middleware
- MVC Architecture
- MongoDB Integration
- Express Routing

---

## 🛠️ Technologies Used

- Node.js
- Express.js
- MongoDB
- Mongoose
- JSON Web Token (JWT)
- EJS
