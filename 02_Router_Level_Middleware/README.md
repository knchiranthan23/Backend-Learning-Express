# Router-Level Middleware in Express.js

## 📖 About

This folder demonstrates the implementation of **Router-Level Middleware** in Express.js using `express.Router()`. Router-level middleware allows middleware functions to be applied only to specific routes, improving code organization and modularity.

---

## 🚀 Concepts Covered

- Creating Router objects
- Using `express.Router()`
- Applying middleware to specific routes
- Route-specific middleware execution
- Organizing routes into separate files

## 📌 Router-Level Middleware

Router-level middleware is created using `express.Router()` and is applied only to the routes associated with that router. It helps organize applications by grouping related routes and middleware into separate modules.

---

## 🎯 Learning Outcome

Through this concept, I learned how to build modular Express.js applications by separating routes into different files and applying middleware only to the required routes using `express.Router()`.
